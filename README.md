Dette er et projekt hvor jeg har brugt en vejr API(open-meteo) til at lave en vejr app der viser lidt info om hvad vejret er på nuværende tidspunkt, samt 3 dage frem i tiden.
Har brugt følgende teknologier:
-React
-Typescript
-SCSS/SASS
-Open-meteo API
